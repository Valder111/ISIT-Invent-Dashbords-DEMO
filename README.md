# ISIT-Invent-Dashbords-DEMO

Публичная демонстрация системы учёта оборудования кафедры ИСиТ: интерактивные дашборды статистики и полнофункциональный интерфейс на **выгруженных данных**. Сервер не требуется - API эмулируется в браузере (MSW + JSON).

**Демо онлайн:** [https://valder111.github.io/ISIT-Invent-Dashbords-DEMO/](https://valder111.github.io/ISIT-Invent-Dashbords-DEMO/)

## Важно

- Все персональные данные и инвентарные номера **отредактированы или вымышленны**.
- Изменения в демо сохраняются в `localStorage` браузера и могут пропасть при очистке данных сайта.
- Это **не** production-система и она не содержит реальной истории разработки приватного репозитория.
